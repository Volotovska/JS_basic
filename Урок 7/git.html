Git - система контроля версий GitHub,GitLab,Bitbucket - бесплатные репозитории для хранения проэктов(хостинг для проэктов) скачать git по ссылке https://git-scm.com/downloads git bush - аналог командной строки для гита на windows 1.git version - команда в терминале(командной строке) для проверки версии git; 2.ls - команда позволяющая посмотреть все папки в директории, в которой мы находимся; 3.cd название папки - команда в терминале(командной строке) для того, чтобы сменить-войти в папку(директорию); 4.git config --global user.name "Имя пользователя" git config --global user.email "Email пользователя" две команды, позволяющие дать гиту понять, что именно этот пользователь с этим email будет вносить изменения в проэкт; 5. git clone ссылка на ваш проэкт - команда, которая позволяет клонировать ваш проэкт с репозитория на ваш локальный компьютер; 6.git add названия файла - позволяет отслеживать изменения в файле; 7.git commit -m "комментарий" - сделать снимок текущего состояния файла и прокоментировать 8.git push -u origin master - позволяет залить файл на репозиторий на ветку мастер 9.git status - показывает отслеживаемые/неотслеживаемые файлы в директории 10.git add . - отслеживать состояние всех файлов в текущей директории; вернуться к слепку;
